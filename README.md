\# IoT Enabled Noise Monitoring System

An IoT-based smart noise monitoring system developed using ESP32 and ESP32-CAM for real-time noise detection, monitoring, and alert generation.



\## 🚀 Features

\- Real-time noise monitoring

\- LCD display output

\- ESP32-CAM integration

\- Wi-Fi monitoring

\- Blynk IoT alerts

\- Buzzer warning system

\- Remote monitoring support



\## 🛠️ Components Used

\- ESP32 DevKit V1

\- ESP32-CAM

\- Sound Sensor

\- LCD Display

\- Buzzer



\## 💻 Software Used

\- Arduino IDE

\- Embedded C

\- Blynk IoT Platform
